    ch395_check_exist

    Description

    Checks if ch395 exists

    Example

     jsr ch395_get_glob_int_status
 cmp #CH395_DETECTED
 beq ch395_connected
 rts
ch395_connected:

    Modify

    • Accumulator

    Returns

    • Accumulator : Returns #CH395_DETECTED if it exists
